# Audio Setup for Food Clicker

Your game now uses REAL audio files instead of synthesized sounds!

## What You Need

1. **Lofi Background Music** (MP3 file)
2. **Bite/Chomp Sound Effect** (MP3 file)

## Where to Download

### Background Music (Lofi)

Pick ONE of these sources and download a lofi track you like:

**Pixabay** (Easiest!)
- Go to: https://pixabay.com/music/search/lofi/
- Browse the tracks and click one you like
- Click "Free Download" button
- Save it as `lofi-music.mp3`

**Chosic**
- Go to: https://www.chosic.com/free-music/lofi/
- Find a track you like
- Download it
- Rename to `lofi-music.mp3`

**ElevenLabs**
- Go to: https://elevenlabs.io/music/lofi
- Browse and download a free track
- Rename to `lofi-music.mp3`

### Bite Sound Effect

**Pixabay Sounds**
- Go to: https://pixabay.com/sound-effects/search/chomp/
- Find a bite/chomp sound you like
- Download it
- Save it as `bite-sound.mp3`

**ZapSplat** (TONS of options!)
- Go to: https://www.zapsplat.com/sound-effect-category/eating-biting/
- Browse eating/biting sounds
- Download one (may need free account)
- Save it as `bite-sound.mp3`

## Tips

- Make sure the file names are EXACT (including the `.mp3` extension)
- If sounds don't play, check the browser console (F12) for error messages
- You can swap out the music anytime by replacing `lofi-music.mp3` with a different file
